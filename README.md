## Requirements
- [Node.js](https://nodejs.org/en/download/) (>= 10.x)
- [CMake](https://cmake.org/download/) (>= 3.15)

## To deploy the application
```bash
# Go into the backend folder
cd cpp-backend
# Run Instal
npm install
# Move back to the main folder
cd ..
# Install the dependencies if not already done
npm install
# Run the code
npm start
```
